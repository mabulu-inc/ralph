## Description

The Ralph Methodology reference is currently copied into the user's project as `docs/RALPH-METHODOLOGY.md` by `ralph init`. Under the built-in-first architecture (T-076), the methodology lives in ralph's code and is injected at runtime. Users extend it via an optional `docs/prompts/methodology.md` file.

This task wires the methodology into the built-in-first prompt assembly and removes it from `ralph init` file generation.

## Changes

- Check for `docs/prompts/methodology.md` in the user's project — if it exists, append its content after the built-in methodology
